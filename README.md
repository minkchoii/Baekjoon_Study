## 🍓 커밋 규칙

- 각자 이름(원하는 대로 ex. witty)을 `branch`로 생성하여 파일을 commit 및 PR한다.
  
- commit 규칙
    - commit 메세지: 문제번호/난이도/문제 주소(URL)
    - 터미널에서 작성법:
    - ```jsx
      git commit -m "1234/silver5/"https://www.어쩌구""
      ```
